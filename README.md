# 知乎收藏夹导出助手

一个用于导出知乎收藏夹内容为Markdown格式的油猴脚本插件。

## 🌟 功能特性

- 📚 **批量导出收藏夹**：支持导出所有收藏夹或选择性导出
- 📝 **Markdown格式**：自动将知乎内容转换为标准Markdown格式
- 🖼️ **图片处理**：自动下载并本地保存图片
- 🔄 **分页处理**：自动处理大量收藏内容的分页加载
- 💾 **本地存储**：支持本地缓存，避免重复下载
- 🧹 **隐私保护**：提供一键清理本地数据功能
- 🌐 **双语支持**：中英文界面和提示

## 🚀 安装使用

### 前置要求
- 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [Greasemonkey](https://www.greasespot.net/) 扩展
- 知乎账号并已登录

### 安装步骤

1. **安装脚本管理器**
   - Chrome/Edge: [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - Firefox: [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)

2. **安装本脚本**
   - 打开 `1.js` 文件，全选复制代码
   - 点击浏览器扩展图标 → 创建新脚本
   - 粘贴代码并保存

3. **验证安装**
   - 访问 [知乎首页](https://www.zhihu.com)
   - 页面右上角应出现"知乎导出工具"面板

## 📖 使用说明

### 基本操作

1. **打开工具面板**
   - 登录知乎后，在任意页面右上角找到"知乎导出工具"面板
   - 首次使用会显示隐私提示

2. **添加收藏夹**
   - 方式一：访问收藏夹页面，工具会自动检测并添加
   - 方式二：手动输入收藏夹URL，点击"添加URL"

3. **选择导出内容**
   - 勾选要导出的收藏夹
   - 选择导出格式（Markdown）

4. **开始导出**
   - 点击"导出选中项"开始批量导出
   - 导出进度会实时显示在状态区域

### 高级功能

#### 📁 文件管理
- **导出位置**：文件默认下载到浏览器默认下载目录
- **文件命名**：自动使用收藏夹名称和日期命名
- **图片处理**：自动创建同名文件夹保存图片

#### 🧹 数据清理
- **清空列表**：清除当前收藏夹列表
- **清除本地数据**：删除所有本地缓存数据
  - ⚠️ 此操作不会影响知乎网站上的实际收藏内容
  - 建议定期清理以保护隐私

#### 🔍 状态监控
- **实时状态**：显示当前操作进度和状态
- **错误处理**：自动处理网络错误和异常情况
- **日志查看**：可在浏览器控制台查看详细日志

## 🔒 隐私安全

### 数据存储
- ✅ 所有数据仅存储在本地浏览器中
- ✅ 不会上传任何个人数据到外部服务器
- ✅ 提供一键清理功能，可随时删除本地数据

### 网络请求
- ⚠️ 脚本会向知乎API发送请求获取收藏夹内容
- ⚠️ 这些请求会在浏览器历史中留下记录
- ✅ 请求仅用于获取用户自己的收藏夹数据

### 隐私建议
- 定期使用"清除本地数据"功能清理缓存
- 在公共电脑上使用后务必清理数据
- 注意浏览器历史记录中可能留下的访问痕迹

## 🛠️ 技术细节

### 核心功能
- 使用 `GM_xmlhttpRequest` 进行跨域请求
- 使用 `GM_download` 下载文件和图片
- 使用 `GM_setValue/GM_getValue` 进行本地存储
- 使用 `DOMParser` 解析HTML内容

### 错误处理
- 网络超时自动重试
- 分页加载失败处理
- 图片下载失败降级处理
- 文件系统权限错误处理
